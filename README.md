Escrowly is a secure escrow payment microservice built to prevent fraud between clients and freelancers.

## Features (MVP)
- User authentication (JWT)
- Contract creation
- Escrow funding
- Work submission
- Approval-based release
- Auto-release after deadline

## Tech Stack
- Spring Boot
- PostgreSQL
- Stripe
- JWT Authentication
- AWS S3 (file storage)

## Architecture
Monolithic for MVP, microservice-ready structure.

## Vision
To become Escrow-as-a-Service API for freelance platforms and agencies.
